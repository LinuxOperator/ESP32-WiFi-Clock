# Changelog

## 1.0.0 - 2026-06-17

- Initial public release.
- WiFi clock firmware for ESP32-C6 Super Mini and TM1637 display.
- Captive portal setup with retry behavior for temporary WiFi outages.
- Web UI for WiFi, display settings, MQTT/Home Assistant, firmware updates, reset, and display test.
- HTTP API for brightness and colon blink.
- MQTT discovery and command/state topics for Home Assistant.
- OTA-ready partition layout with persistent NVS settings.
